# Student Life Management System

## Overview
This is a **Java console-based program** designed to manage a student's life in a simplified system.  
It allows you to track student details, courses, grades, and extracurricular activities.

---

## Features
- Add a new student
- View all students
- Add courses for a student
- Add extracurricular activities
- Add grades
- Search for a student by ID
- Update student information

---

## How to Run
1. Make sure you have **Java installed**.
2. Save the code as `StudentLifeManager.java`.
3. Open a terminal or command prompt.
4. Compile the program:

```bash
javac StudentLifeManager.java
